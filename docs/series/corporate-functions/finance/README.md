# 재무 — Finance

회사의 자금 흐름, 자원 배분, 재정 건강을 관리하는 시스템을 소프트웨어 엔지니어링의 관점에서 분석합니다.

## 도메인 개요

재무는 회사의 모든 활동에 대한 비용을 추적하고, 자원을 배분하며, 성과를 측정합니다. 마치 소프트웨어의 성능 프로파일링과 리소스 할당과 유사합니다.

## 주요 은유

- **예산**: Resource Allocation — 회사 자원의 전략적 배분
- **비용**: Runtime Cost — 각 활동 실행에 드는 비용
- **회계마감**: Batch Closing Cycle — 주기적 재정 정산과 보고
- **성과 분석**: Profiling — 어느 부문이 효율적인지 분석
- **ROI**: Return on Investment — 투자 대비 효과의 척도
