# 법무 — Legal

회사의 법적 위험 관리와 규정 준수를 위한 시스템을 소프트웨어 엔지니어링의 관점에서 분석합니다.

## 도메인 개요

법무 팀은 회사를 법적 위험으로부터 보호하고, 계약을 통해 외부 조직과의 인터페이스를 정의합니다. 마치 소프트웨어의 보안, 테스트, 코드 리뷰의 역할입니다.

## 주요 은유

- **계약**: Interface Contract — 양 조직 간의 약속과 책임 정의
- **컴플라이언스**: Rule Set — 따라야 할 법적 규칙과 정책
- **분쟁**: Conflict Resolution — 문제 해결과 법적 소송
- **계약 검토**: Code Review — 계약 체결 전 법적 검증
- **조항**: Function Parameter — 계약의 각 조건 요소
