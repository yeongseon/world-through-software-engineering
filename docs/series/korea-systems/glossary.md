# 대한민국 시스템 용어 사전

이 문서는 대한민국 시스템 시리즈에서 사용하는 도메인 특화 용어들을 정리합니다.

## 정치 (Politics)

| 실제 용어 | 메타포 | 설명 |
|----------|--------|------|
| 헌법 | Root Architecture | 모든 다른 법과 제도의 기반이 되는 법 |
| 법률 | API Specification | 국회가 제정하는 상위 수준의 규칙 |
| 시행령 | Implementation Detail | 대통령령, 법을 시행하기 위한 구체적 규칙 |
| 국정감사 | System Audit | 국회가 행정부의 업무를 감시하고 검증하는 프로세스 |
| 선거 | Leadership Rotation | 주기적인 리더십 교체 메커니즘 |
| 국회의원 | Process Handler | 입법 권한을 행사하는 주요 행위자 |
| 교섭단체 | Major Process Group | 국회에서 일정 수 이상의 의원을 가진 정당/단체 |

## 법률 (Law)

| 실제 용어 | 메타포 | 설명 |
|----------|--------|------|
| 법체계 | Layered Architecture | 헌법→법률→시행령→규칙의 계층적 구조 |
| 위임입법 | Dependency Injection | 상위 법이 하위 법에 세부 규칙 제정권 위임 |
| 법개정 | Changelog | 법의 변경 이력과 버전 관리 |
| 위헌심사 | Validation | 법이 헌법 원칙을 위반하지 않는지 검증 |
| 판례 | Design Pattern | 반복되는 법적 상황의 해결 방식 |
| 법원 | Validation Engine | 법을 해석하고 적용하는 시스템 |
| 상소 | Appeal/Retry | 이전 판결에 대한 재검토 및 재판 |

## 경제 (Economy)

| 실제 용어 | 메타포 | 설명 |
|----------|--------|------|
| 예산 | Resource Allocation | 국가 재정의 배분 및 우선순위 결정 |
| 세금 | Incentive Mechanism | 경제 행동에 영향을 주는 신호 |
| 금리 | Control Parameter | 경제를 조절하는 주요 변수 |
| 전세 | Unique Protocol | 한국만의 고유한 부동산 거래 방식 |
| 부동산 | Asset System | 장기 자산 저장 및 가치 이전의 프로토콜 |
| 인플레이션 | System Drift | 시간이 지남에 따라 누적되는 시스템 변화 |
| 환율 | Interface Bridge | 국내외 경제 시스템을 연결하는 변환 요소 |

## 사회 (Society)

| 실제 용어 | 메타포 | 설명 |
|----------|--------|------|
| 연금 | Sustainability System | 장기적 재정 지속성을 위한 시스템 |
| 교육 | Sorting Pipeline | 인재 발굴, 선별, 분류하는 프로세스 |
| 사교육 | Shadow Processing | 공식 시스템 외부의 병렬 처리 |
| 출산율 | Throughput Indicator | 사회 시스템의 주요 성능 지표 |
| 보육 | Early Stage Pipeline | 미래 인력의 초기 준비 단계 |
| 주택 | Infrastructure | 사회의 기본 인프라이자 자산 저장소 |
| 의료 | System Reliability | 사회의 가용성(availability)을 유지하는 시스템 |

| 공청회 | Community Review | 새로운 정책이나 법안에 대한 공개 의견 수렴 |
