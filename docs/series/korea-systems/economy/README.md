# 경제 — Economy

대한민국의 경제 시스템을 소프트웨어 엔지니어링의 관점에서 분석합니다. 자원 할당, 인센티브 메커니즘, 제어 변수, 그리고 한국 고유의 부동산/전세 제도를 시스템 설계로 이해합니다.

## 도메인 개요

경제 시스템은 자원 배분과 가치 이동을 조율하는 복잡한 시스템입니다. 소프트웨어 시스템처럼 인센티브, 제어 변수, 고유 프로토콜이 있습니다.

## 주요 은유

- **예산**: Resource Allocation — 국가 재정의 전략적 배분
- **세금**: Incentive Mechanism — 경제 행동에 영향을 주는 신호
- **금리**: Control Parameter — 경제를 조절하는 주요 변수
- **전세**: Unique Protocol — 한국만의 고유한 부동산 거래 방식
- **부동산**: Asset System — 장기 자산 저장과 가치 이동 시스템
