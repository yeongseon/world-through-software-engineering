# 사회 — Society

대한민국의 사회 제도를 소프트웨어 엔지니어링의 관점에서 분석합니다. 연금, 교육, 출산율, 사교육을 시스템 성능 지표와 아키텍처로 이해합니다.

## 도메인 개요

사회 제도는 장기적 지속성, 인재 선발, 성과 지표를 관리하는 복잡한 시스템입니다. 소프트웨어 시스템의 가용성, 파이프라인, 성능 지표와 유사한 구조를 가집니다.

## 주요 은유

- **연금**: Sustainability System — 장기적 재정 지속성 메커니즘
- **교육**: Sorting Pipeline — 인재 발굴, 선별, 분류 프로세스
- **출산율**: Throughput Indicator — 사회 시스템의 핵심 성능 지표
- **사교육**: Shadow Processing — 공식 시스템 외부의 병렬 처리
- **보육**: Early Stage Pipeline — 미래 인력의 초기 준비 단계
